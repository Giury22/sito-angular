Home

Elemento Tag HTML Posizione / Comportamento

Carosello immagini img + button Prev + button Next Scorrimento automatico a intervalli. Frecce laterali per navigazione manuale.
Indicatori carosello (pallini) button (uno per slide) Sotto il carosello. Click salta direttamente alla slide corrispondente. Pallino attivo evidenziato.

Statistiche museo (×3) div.stat > span (valore) + span (etichetta) Tre blocchi affiancati: visitatori/anno, pezzi in collezione, anni di storia.

Collezione

Elemento Tag HTML Posizione / Comportamento
Player video video[controls] In cima alla pagina. Filmato introduttivo del museo. Controlli nativi browser.
Campo ricerca input[type="search"] Filtra le card in tempo reale mentre si digita.
Filtri chip per decade (×3) button (1960s, 1970s, 1980s) Filtrano le card per decade di produzione. Un chip attivo alla volta; ri-cliccando si deseleziona.
Slider anno di produzione input[type="range"] Mostra solo i modelli prodotti fino all'anno selezionato. Min 1960, max 1980, aggiornamento in tempo reale.
Card modello Vespa (×7) div.col-card Griglia di 7 card, una per modello. Contiene gli elementi qui sotto.
   • Fotografia modello img Cliccabile per aprire il lightbox. Cursore zoom-in.
   • Nome, anno, cilindrata p (×2) Intestazione della card sempre visibile.
   • Toggle switch "Ulteriori info" button Apre/chiude il pannello dei dettagli. Più card possono essere aperte contemporaneamente.
   • Descrizione + dati tecnici p (×2) Visibili solo a card aperta. Testo bilingue (IT/EN).
   • Link Wikipedia a[target="_blank"] Visibile solo a card aperta. Apre Wikipedia in nuova scheda.
Messaggio "Nessun risultato" p Compare quando nessuna card corrisponde ai filtri attivi.
Lightbox immagine div overlay + img + button chiudi Click sulla foto apre un overlay fullscreen con l'immagine ingrandita, nome e dati del modello. Tasto ESC chiude.

Biglietti

Elemento Tag HTML Posizione / Comportamento
Tabella prezzi table con caption, thead, tbody 3 righe: Intero, Ridotto, Gratuito. Colonne: tipo, descrizione, prezzo.
Navigazione mese (Prev / Next) button (×2) + span titolo mese Scorre il calendario al mese precedente o successivo.
Calendario disponibilità Griglia div a 7 colonne CSS Giorni del mese con stati: disponibile (verde), quasi esaurito (arancio), esaurito (rosso), chiuso (grigio). Click su giorno disponibile lo seleziona.
Legenda colori calendario div con span (×4) Sotto il calendario. Spiega i colori degli stati.
Input data nativa input[type="date"] Sincronizzato col calendario custom. Attributi min e max impostati.
Avviso data non disponibile span inline Compare quando si sceglie una data chiusa o esaurita tramite il picker nativo.
Select fascia oraria select 4 opzioni di orario (10-12, 12-14, 14-16, 16-18).
Checkbox + contatore quantità (×3) input[type="checkbox"] + input[type="number"] Una riga per categoria (Intero, Ridotto, Gratuito). Il contatore appare solo se la checkbox è spuntata.
Totale dinamico div con strong Totale in euro. Appare solo con almeno una categoria selezionata, aggiornamento in tempo reale.
Pulsante "Prenota" button Valida il form e, se corretto, mostra il toast di conferma.
Toast di conferma prenotazione div overlay Overlay che copre la pagina con titolo, messaggio di conferma e pulsante "Chiudi" che resetta il form.
